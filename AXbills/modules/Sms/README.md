# Добавление SMS шлюза MTS (Беларусь)
```
$conf{SMS_MTSSMS_URL}='https://api.communicator.mts.by/';
$conf{SMS_MTSSMS_LOGIN} = '';
$conf{SMS_MTSSMS_PASSWORD} = '';
$conf{SMS_MTSSMS_ALPHA_NAME}='';
$conf{SMS_MTSSMS_CLID}='';

$conf{SMS_MTSSMS_EXT}='';
$conf{SMS_MTSSMS_SENDER}='';
$conf{SMS_MTSSMS_DEBUG}=3;
$conf{SMS_DEBUG}=1;
```
